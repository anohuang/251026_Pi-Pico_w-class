# 安裝uv環境，參考老師REPO的python下的UV步驟

# 1. uv init --python 3.10  
  此語法是用來建立pyproject.toml和.python-version檔案的 (--後面是指定要裝的python版本)
# 2. uv venv  
  此語法是用來建立.venv隱藏檔的，作為虛擬環境之用
# 3. uv sync  
  此語法是將pyproject.toml中的說明與外掛等資料同步至.venv中
# 4. uv add
  add後面空一格輸入想要安裝的外掛套件，即可自動安裝並同步
# 5 .uv remove
  remove後面空一格輸入想要移除的外掛套件，即可自動移除並同步
# 6. .py是真正的專案檔  .ipynb是學習和測試檔
  需要安裝ipykernel外掛和延伸需要安裝ipykernel外掛和延伸模組jupyt模組jupyter)
# 7. 寫程式時每一行都應該要是敘述式(執行時不會跳出結果)，不應該出現運算式(執行後會跑出結果)
